==CONCEPTS==

display: flex;

faz com que todos os elementos selecionados fiquem inline e mudando o tamanho x e y para o tamanho apenas do conteúdo

==DIRECTION==

flex-direction: row; (padrão)
column; (um abaixo do outro)
row-reverse; (inverte os eixos e começa do lado contrário do padrão)
column-reverse; (inverte os eixos e fica de baixo para cima)

==ALIGNMENT & SPACES==

Alinhamento dentro do container e distribuição dos elementos no eixo principal
justify-content: flex-start; (padrão)
flex-end;
center;
space-evenly; (espaços constantes, )
space-around; (espaços ao redor)
space-between; (espaços entre os elementos)

Para funcionar as propriedades de espaçamento do justify-content em flex-direction: column; devemos definir um height. O row não tem necessidade pois trabalha na largura(width) em elementos como div que ja vem com largura de uma extremidade da página até a outra, então vai trabalhar nessa extremidade quando for row.


Alinhamento no eixo cruzado. Para usar o align-items devemos definir uma altura caso o flex-direction seja row.

align-items: stretch; (estica/padrão)
flex-start;
flex-end;
center;
baseline;

Espaço entre elementos. Valores podem ser relativos, fixos ou flexíveis 
gap: 10px/em/%

margin: auto;

==MULTI-LINE==

Serve para você criar multi-linhas e usar também o align-content para alinhar os conteúdos.

flex-wrap: wrap;
