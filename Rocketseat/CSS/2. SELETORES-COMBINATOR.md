SELETORES

- type | element | tag
- id #
- class .
- atributo [title] ou [title="t"]
- universal *

COMBINATOR

- Descendent = article p {}
- List = p, h1, body {}
- Next sibling = h2 + p {}
- Child = aside > ul {} //neste caso ele pega o primeiro elemento ul filho que tiver em aside, ou seja, se tiver vários ul dentro de aside, ele só vai pegar o primeiro
